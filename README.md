## vue3-tiptap-editor 

vue3-tiptap-editor是基于最新tiptap框架开发的vue3富文本组件，具有放置xss工具，插入图片，插入表格等功能。

### 安装依赖
#### 在pnpm安装本地依赖
pnpm add @majoreditor/ui
测试成功

#### pnpm 安装依赖到dev环境
```js
pnpm add -D vite-plugin-svg-icons -w
```

#### 更新所有包依赖
pnpm install --recursive

### 运行
```js
pnpm dev
or
npm run dev
```

### 打包
```js
pnpm build
or
npm run build
```
